# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Please read the full text at the link above. All participants in this project are expected to uphold this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, please report it by opening a GitHub issue or contacting the maintainers directly.
